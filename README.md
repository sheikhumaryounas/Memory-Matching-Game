# 🧠 Memory Matching Game

A simple and interactive **Memory Matching Game** built using **HTML, CSS, JavaScript, and jQuery**.  
The objective of the game is to match all pairs of cards in the least number of moves.

## 🎮 Game Features

- 🃏 4x4 grid memory board  
- 🔀 Random card shuffle every time  
- 👆 Click to flip cards  
- ✅ Matching pair detection  
- 🔒 Board lock while checking match  
- 🔁 Restart game button  
- 📊 Move counter tracker  
- 🎨 Modern UI with gradient design and smooth animations  

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Grid, Flexbox, Animations, Gradients)  
- JavaScript  
- jQuery 3.6.0

## 🚀 How to Run the Game

1. Download or clone this repository:


git clone https://github.com/sheikhumaryounas/Memory-Matching-Game.git


2. Open the project folder.  
3. Double-click `index.html`.  
4. Play the game in your browser.  

## 🎯 How to Play

1. Click on any card to flip it.  
2. Click another card to try and find its matching pair.  
3. If the symbols match, the cards stay flipped ✅  
4. If they do not match, they flip back after 1 second 🔄  
5. Match all pairs with the minimum number of moves 🏆  

## 🔄 Restarting the Game

Click the **Restart Game** button to:  
- Reset moves to 0 🔢  
- Shuffle the board 🔀  
- Start a new game 🆕
